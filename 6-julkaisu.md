# Vercel julkaisu (html+js)

## 1. Vie projekti GitHubiin

`index.html` toimii aloitussivuna.

## 2. Julkaise Vercelissä

Luo Vercel-tili https://vercel.com/signup
Mene osoitteeseen https://vercel.com/dashboard

- Klikkaa “Add New → Project”

- Valitse “Import Git Repository”

- Valitse GitHub-reposi, jossa on HTML-projektisi

- Vercel tunnistaa projektin ja ehdottaa asetuksia (yleensä sopivia)

- Paina Deploy

Hetken kuluttua saat **URL**in.

## Automaattinen päivitys

Jos muutat koodia ja teet uuden git pushin, Vercel julkaisee automaattisesti uuden version.
